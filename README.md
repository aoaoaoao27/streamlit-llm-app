# streamlit-llm-app

## 概要
このアプリは、Streamlitを使用して構築されたWebアプリケーションで、LLM（大規模言語モデル）を活用してユーザーの質問に答えます。異なる専門分野（例: 医療、法律）に基づいて、適切な応答を提供します。

## 機能
- **医療の専門家モード**: 医療の専門家として振る舞い、患者の症状に基づいて診断や治療法を提案します。
- **法律の専門家モード**: 法律の専門家として振る舞い、法的な質問に対してアドバイスを提供します。

## 使用方法
1. アプリを起動します。
2. 動作モード（医療の専門家または法律の専門家）を選択します。
3. 質問を入力し、「Submit」ボタンを押します。
4. 選択したモードに基づいて、LLMが応答を生成します。

## 必要な環境
- Python 3.8以上
- 必要なPythonライブラリは`requirements.txt`に記載されています。

## インストール
1. リポジトリをクローンします。
   ```bash
   git clone https://github.com/aoaoaoao27/streamlit-app-deploy.git
   ```
2. 必要なライブラリをインストールします。
   ```bash
   pip install -r requirements.txt
   ```

## 実行方法
以下のコマンドでアプリを起動します。
```bash
streamlit run app.py
```

## ファイル構成
- `app.py`: アプリケーションのメインスクリプト。
- `README.md`: このファイル。
- `requirements.txt`: 必要なPythonライブラリ。
